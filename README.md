# Wishlist — Список желаний

Веб-приложение для управления списком желаний на Flask.

## Возможности
- Добавление, редактирование, удаление желаний
- Поиск по названию и ссылке
- Сортировка по цене и статусу
- Подсчёт общей стоимости

## Технологии
- Flask (Python)
- SQLite
- pytest

## Запуск
```bash
pip install -r requirements.txt
python app.py
